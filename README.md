# SplitBox
Split your screen and create complex layouts with resizeable panes.

Example layout:
```html
<hbox>
    <div>
        Top content
    </div>
    <vbox>
        <div>
            bottom left content
        </div>
        <hbox>
            <div>bottom right - top section</div>
            <div>bottom right - bottom section</div>
        </hbox>
    </vbox>
</hbox>
```
